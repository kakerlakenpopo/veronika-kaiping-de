CREATING A PUBLIC, SCULPTURAL AND MULTIFUNCTIONAL SPACE.


ORT – social design

44FLAVOURS@DESSAU, a two week project with Prof. Brigitte Hartwig featuring Sebastian Bagge and Julio Rölle from 44flavours.

The project was about creating structures and sculptures in a public space. After the first meeting with the 44flavours we figured it better to create on sculptural object as a group rather that in small teams or by ourselves. We wanted it to be still useful and costing nearly nothing by using almost only recycled materials.
When exploring the space that we could build upon the decision to create a communual, multifunctional space was made and the very same day we started building it. The building process took 2 days with the tasks split evenly. We documented the process in a booklet, created a small paper model for setting up yourself and prepared a inauguration party.

We exhibited the project with our university at the designers open in Leipzig 2017.
